This directory contains third party jars and binaries required for integration with some miners and calculators. They are excluded from this public repository for reasons of space and redistribution rights. The listing in a full install has the following structure:

$ find ldlib/lp_solve_5.5_java/
ldlib/lp_solve_5.5_java/
ldlib/lp_solve_5.5_java/lib
ldlib/lp_solve_5.5_java/lib/lpsolve55j.jar
ldlib/lp_solve_5.5_java/lib/win64
ldlib/lp_solve_5.5_java/lib/win64/lpsolve55.dll
ldlib/lp_solve_5.5_java/lib/win64/lpsolve55j.dll
